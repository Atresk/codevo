# 練習題 2：交換變數的值

> 本題可用 Python、JavaScript、C++ 任一語言完成

---

## 📘 題目描述

請撰寫一支程式，交換兩個變數的值。

1. 先將變數 `a` 設為 `10`，`b` 設為 `25`
2. 交換兩個變數的內容
3. 輸出交換後的結果：

```
After swap:
a = 25
b = 10
```

## 📤 輸出範例

```

```

## 💡 提示

* 可使用「暫存變數」來交換值
* Python 可用 tuple unpacking：`a, b = b, a`
* JS 可用解構賦值：`[a, b] = [b, a]`
* C++ 可使用暫時變數或標準交換技巧
